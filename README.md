# Xerar datos ficticios (Tarefa UD2)

## Descripción
 Xerar e almacenar nun dicionario datos ficticios de 15 usuarios.

* Nome
* Dirección
* Correo electrónico
* Teléfono

Mostrar os datos dos usuarios e o nome dun afortunado aleatorio.

## Requisitos
- Python 3.x
- Contorno virtual (recomendado)
- Paquete Faker
